# Willow Server

Willow server api.

```
Willow API

Options:
  --clients   Client origin (enable CORS)                         [default: "*"]
  --inactive  Tracer maximum inactive time (minutes)               [default: 10]
  --lifetime  Tracer maximum live time (minutes)                   [default: 30]
  --port      Set the server port                                [default: 8000]
  --secret    The secret for client sessions                 [default: "secret"]
  --tracers   Address of the tracer server                   [string] [required]
  -h, --help  Show help                                                [boolean]
```
